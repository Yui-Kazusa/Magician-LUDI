# Magician-LUDI
撸弟法师の变强之路
wasd移动 按住shift疾跑 鼠标指向攻击方向
左键为发射火球 右键为流星火雨 Q键为火龙卷

左上角为血量条 魔法值 攻击耐力值（只与左键攻击有关，为0则不可用左键攻击防止连死）
正下方为经验值 满了会升级

技能介绍：
火球：单发攻击，伤害较高，且具备打断敌人攻击造成硬直的效果，硬直时长在半秒内随机，同时也可以和敌人远程攻击counter

流星火雨：无视距离的从天而降的范围攻击，每个火球攻击伤害不定，也具有硬直效果，下落位置为鼠标在屏幕上的位置

火龙卷：控制加聚怪技能，释放一个火焰龙卷风向鼠标方向缓慢移动，会吸附除了boss以外的敌人，并在敌人靠近中心时持续造成伤害，同样具有硬直效果，移动距离有限制，能够counter所有攻击（火龙卷+火球能连

P.S. 你的魔法值，所有攻击值，攻击耐力值都会随着等级上升而提高，打不过就多刷吧，并且每升一级所有状态全部回满

敌人：
红色小怪：最弱的小怪，只会使用近战攻击，伤害较低，血量较少，移速较快，杀死经验值+10
白色精英怪：只会使用近战攻击，血量中等，移速一般，伤害一般，杀死经验值+30
黄色精英怪：只会使用远程攻击，血量较高，移速较慢，伤害一般，
为单方向发射扇状雷电球攻击，杀死经验值+50
紫色BOSS：只会使用远程攻击，血量高，移速一般，伤害高，攻击具有三个阶段与BOSS血量相关，一阶段为三重冰球扇状攻击弹速较快，二阶段为360度雷电球环绕攻击，中间有随机空隙，弹速较慢，三阶段为冰球加雷电球组合攻击。

P.S. 跑出敌人视野就不会追你，只要攻击敌人敌人就会追你且移速翻倍

本游戏一共三个关卡：
关卡一具有较多障碍物可以躲避攻击，关卡二障碍物稀少，关卡三为SSVGG

道具：
本游戏一共有五种道具：小回蓝，大回蓝，小回血，大回血，以及？？？（自己去试吧，小怪只会有几率掉落小道具，精英怪只会有几率掉大道具，BOSS百分百掉落大道具

本游戏一共九首BGM
